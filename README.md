## 5. Must-Master Topics

### Beginner

* Arrays
  n- Sorting
* Prefix Sum
* Binary Search
* Two Pointers
* Greedy
* Recursion / Backtracking

### Intermediate

* Graphs (BFS, DFS, Topological Sort)
* Dijkstra
* DSU
* Trees
* Basic DP
* Number Theory (GCD, Sieve, Modular Arithmetic)

### Advanced

* Segment Tree
* Fenwick Tree
* Lazy Propagation
* LCA
* SCC
* Max Flow / Min Cut
* Matching
* Bitmask DP
* Digit DP
* Convex Hull Trick
* FFT / NTT
* Strings (KMP, Z, Trie, Suffix Array)
* Geometry

---

## 6. Training Plan

### Daily (1–3 hours)

* Solve 2 medium problems OR 1 hard problem
* Read editorial after attempts
* Review one old concept

### Weekly

* 2 live contests minimum
* Upsolve every unsolved contest problem
* One topic-focused study session

### Monthly

* Mock ICPC team contest
* Review progress and weak topics

---

## 7. Milestones

### Stage 1

* Finish CSES Intro + Sorting + Graph basics
* Reach 200 solved problems

### Stage 2

* Reach Codeforces Specialist / Expert
* Finish AtCoder DP Contest
* Solve 500+ problems

### Stage 3

* Solve old ICPC regionals consistently
* Build team strategy
* Reach Candidate Master+
